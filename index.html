GitHub

GitHub commands manual for the Terminal

  1. To add a file - which will send the file to staging area

    git add "file name"

  2. To check status

    git status

  3. To commit changes comment

    git commit -m "changes comment"

  4. To publish in the GitHub

    git push origin main

  5. To delete a file

    git add "file name"

  6. To get update changes made in GitHub file - Download

    git pull

  7. To upload the file/changes made in repository

    git init

    git remote add origin

  8. To add bulk folder/files - which moves to staging area

    git add.

    git commit -m "change comments"

  9. To change directory folder

    cd ..

    cd "file name"

  10. Create branch - master

    git branch

    git push origin master

  11. Change the name of branch

    git branch -m "folder name"

  12. To check all the branches

    git branch -a

    git pull

  13. To get inside of the branch folder

    git checkout "file name"

  14. To check branch name

    git branch -a

  15. To check the difference between the main and branch

    git diff "file name"

  16. To merge the file with main

    git merge "file name"

    git push origin main

  17. Pull request raising to admin

    It can be requested in the website - create pull request

  18. Clear all in terminal

    clear
